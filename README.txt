The PawBoost Lost and Found Pets plugin allows you to display a list of lost & found pets as a widget on your WordPress site.

== Description ==
The PawBoost Lost and Found Pets plugin allows you to display a list of lost & found pets as a widget on your WordPress site. Help create more happy reunions by raising local awareness for lost pets.

Choose the status (lost, found, or lost & found), location, species, and number of pets per page you want to feature. Sort pets by recency or distance from the chosen location.

== Installation ==
1. Upload the plugin files to the `/wp-content/plugins/plugin-name` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the \\\'Plugins\\\' screen in WordPress.
3. Go to Appearance -> Widgets, then under Available Widgets drag & drop PawBoost Lost and Found Pets to the Widget Area where you want it to appear on your site
4. Click PawBoost Lost and Found Pets to expand the configuration menu, then configure it how you wish
